## Component Responsibilities

| Component | Responsibility |
|-----------|----------------|
| `Card` | Represents a playing card (suit, rank, value) |
| `Deck` | Manages card shuffling and dealing |
| `Hand` | Manages player/dealer cards and score calculation |
| `Player` | Player entity with betting capabilities |
| `Dealer` | Dealer with fixed playing rules |
| `GameEngine` | Core game logic and state management |
| `UI` | Text-based user interface |

## Game Flow

```
START вҶ’ DEAL_INITIAL_CARDS вҶ’ PLAYER_TURN вҶ’ DEALER_TURN вҶ’ DETERMINE_WINNER вҶ’ END/REPLAY
```
